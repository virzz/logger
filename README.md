# Logger

## Import 

```golang
import "github.com/virzz/logger"

logger.Debug("Debug")
logger.DebugF("DebugF")
logger.Success("Success")
logger.SuccessF("SuccessF")
logger.Error("Error")
logger.ErrorF("ErrorF")
logger.Warn("Warn")
logger.WarnF("WarnF")
logger.Info("Info")
logger.InfoF("InfoF")
logger.Print("Print")
logger.Printf("Printf")

```
